უნდა გვეყურებინა ვიდეოსვის და კარგად დაგვემახსოვრებინა css positions: 1. static - ეს არის ელემენტის მახასიათებელი, თუ სხვა პოზიცია არ არის მითითებული. 2. relative - ელემენტი იცავს თავის ბუნებრივ ადგილს, მაგრამ შემდეგ შეიძლება გადაიტანოს სხვა ადგილას top, right, bottom, left პროპერტეებით. relative-ის გამოყენება ხელსაყრელია სხვა ელემენტების მიმართ ადგილმდებარეობის გამოსაწორებლად. 3. absolute - ელემენტი პოზიციონირდება მისი ყველაზე უახლოესი პოზიციონირებული (relative, fixed, absolute) პარის ელემენტის მიმართ. მას შეუძლია გადალახოს გვერდის ნაკადი და დაეტოვოს თავის ადგილს. 4. fixed - ელემენტი მიმართულია მთელი ეკრანის მიხედვით და არა მისი მშობლიური ელემენტის. 5. sticky - ელემენტი თავდაპირველად მუშაობს როგორც relative, მაგრამ როდესაც პანელი გადადის განსაზღვრულ ადგილას (scroll-ზე), ის ხდება fixed პოზიციაში.
